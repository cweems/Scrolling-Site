Introduction

This is a simple test of a Parallax JavaScrips site.

This slide will demonstrate the types of effects that can be implemented with scrolldeck.js.


Javascript allows for the creation of dynamic content that changes in tandem with a user's interaction with the site. For example, this slide offsets the background during a scroll action in order to create a parallax effect.
